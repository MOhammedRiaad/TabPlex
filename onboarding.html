Quick Start Guide

  1. Open TabBoard - Click the extension icon or press Ctrl+Shift+T
  2. Organize Tabs - Go to Boards view, create folders, and drag tabs to organize
  3. Create Tasks - Use Today view to add tasks with due dates and priorities
  4. Take Notes - Write markdown notes linked to your boards or tasks
  5. Track Focus - Use the Pomodoro timer in Sessions view to track work time
  6. Explore - Try Bookmarks, History, Canvas, and Analytics to discover more
